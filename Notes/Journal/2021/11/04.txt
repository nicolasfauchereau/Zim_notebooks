==== convert GPX files to KML for display in Google Maps ====

@adventure 

1) sudo apt-get install gpsbabel-gui

then 


gpsbabel -w -i gpx -f Otorohaea_Trig_Road_250.gpx -o kml -F Otorohaea_Trig_Road_250.kml

to import in google maps, go to 

https://www.google.com/maps/d/?hl=en 


	Open Google My Maps.
	Create a new map.
	Press import into the upper left corner.
	Now open the KML file. Or drag the KML file into the import window.
	You did it! You have successfully imported a KML into Google Maps.
